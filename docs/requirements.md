# Business File Automation Suite

## Project Objective

Build a desktop application that automates repetitive file management tasks for businesses and individuals.

---

## Target Users

- Small Businesses
- Office Employees
- Students
- Teachers
- HR Teams
- Accountants
- Content Creators

---

## Version 1 Features

- Bulk Rename Files
- Add Prefix
- Add Suffix
- Replace Text
- Remove Text
- Change Letter Case
- Organize Files by Type
- Generate Processing Report

---

## Future Features

- Duplicate File Finder
- Empty Folder Cleaner
- PDF Tools
- ZIP Tools
- AI File Categorization
- Image Compression

---

## Technologies

- Python
- Tkinter
- pathlib
- shutil
- logging
